### Proyectos Destacados

#### 1. eva-oa — Entorno Virtual de Aprendizaje
Una plataforma de e-learning fullstack con una clara separación de responsabilidades.
- **Arquitectura:** API RESTful en **Express** que gestiona la lógica de negocio y los datos, desacoplada de un cliente SPA en **React**.
- **Autenticación:** Sistema basado en roles (estudiante, profesor, admin) con JSON Web Tokens (JWT), asegurando los endpoints y el acceso a los recursos.
- **Features:** Gestión completa de cursos (CRUD), sistema de inscripción, y foros de discusión para la interacción.

#### 2. plan-comunitario-app — Sistema de Gestión de Riesgos
Aplicación monolítica que digitaliza el Plan Comunitario de Gestión de Riesgos de Desastres, construida sobre **Lumen**, el micro-framework de Laravel.
- **Arquitectura:** Patrón MVC para organizar el código. El ORM Eloquent gestiona la persistencia de datos en MySQL.
- **UI:** Vistas tradicionales renderizadas en el servidor con Blade y estilizadas con Bootstrap, ofreciendo una interfaz administrativa rápida y funcional.
- **Scope:** Cubre el ciclo de vida completo de la gestión de riesgos, desde la identificación de amenazas y vulnerabilidades hasta la matriz de recursos comunitarios.

#### 3. ashcroft08.github.io — E-commerce Frontend
Demostración de que no siempre se necesita un framework. Un e-commerce de presentación construido con **HTML, CSS y JavaScript puro**.
- **Performance:** Carga rápida y sin dependencias, ideal para un sitio estático desplegado en GitHub Pages.
- **Interactividad:** Lógica de UI y animaciones implementadas en JavaScript vanilla, mostrando un entendimiento profundo de los fundamentos del DOM.
